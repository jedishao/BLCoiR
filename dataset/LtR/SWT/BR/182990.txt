Bug 182990  NullPointerException in monitorEnumProc() in Class DisplayDescription:

The method getMonitors is not reentrant if multiple user interface threads are used. The variable 'monitors' is static and could be set to null by an other thread while one thread still executes getMonitors. This will result in a NullPointerException.
